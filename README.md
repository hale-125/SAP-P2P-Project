# SAP Procure-to-Pay (P2P) Project

This project demonstrates the end-to-end Procure-to-Pay (P2P) cycle using SAP MM and SAP FI.

## Process Flow
Purchase Requisition → Purchase Order → Goods Receipt → Invoice Verification → Payment

## Description
The project simulates a real-world procurement process in a manufacturing company. It covers all stages from material request to vendor payment.
